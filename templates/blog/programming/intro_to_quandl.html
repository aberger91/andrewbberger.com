Quandl Databases - Getting Started 
 Andrew Berger Oct. 15, 2016
Quandl is a web service that provides free databases for financial, economic, demographic, and environmental information. It's really easy to access quandl.com with a variety of languages (Excel, R, Ruby) and free to use.

If you don't have the Python API, don't worry - just pip install quandl .

Quandl offers a 'Database Browser' where you can choose from stock, futures, commodity, currency, option, economic, industry data, and more - from the United States and the rest of the world! For futures data, there are databases for the Chicago Mercantile Exchange (CME), Intercontinental Exchange (ICE), the CBOT, and more.

Quandl is also well-documented You do not need any special software to use the Quandl API. You can paste any API call directly into your browser address bar, to retrieve data.

You can also make API calls from the command line (console / terminal) of your computer, using curl or wget or similar utilities.

Authenticated users have a limit of 300 calls per 10 seconds, 2,000 calls per 10 minutes, and a limit of 50,000 calls per day. Premium data subscribers have a limit of 5,000 calls per 10 minutes, and a limit of 720,000 calls per day. Anonymous requests are subject to a limit of 50 calls per day. Anonymous requests will be deprecated soon - so you should make an account!
